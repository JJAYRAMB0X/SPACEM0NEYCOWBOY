🔍 Search Tips
₿ Crypto
  • Search by symbol — BTC, ETH, SOL
  • Search by name — Bitcoin, Monero
  • Paste a contract address for exact lookup
  • Results from DexScreener & CoinGecko
  • If a result doesn't add on first click, try clicking it again
📈 Stocks
  • Company name works best — "Air Canada"
  • Ticker also works — AAPL, MSFT, AC
  • US stocks via Finnhub
  • Canadian & international via Twelve Data
  • If one result fails, try a different one
  • If a result doesn't add on first click, try clicking it again
✏️ Manual Entry
  • Use + Manual if search doesn't find what you need
  • Add Crypto, Stocks, Precious Metals, or Cash manually
  • Right-click any asset → Edit Info to update price later
